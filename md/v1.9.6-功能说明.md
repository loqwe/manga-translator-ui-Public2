# v1.9.6 功能更新说明

## 新功能

### 1. MangaJaNai 超分模型支持

- **新增 17 个 MangaJaNai 超分模型**：
  - **x2 模型**：7个不同分辨率的模型（1200p至2048p）
  - **x4 模型**：7个不同分辨率的模型（1200p至2048p）
  - **DAT2 x4 模型**：3个模型，包括IllustrationJaNai

- **自动模型选择**：
  - 根据输入图片分辨率（1200p-2048p）自动选择最合适的模型
  - 分辨率区间划分：
    - 小于1250px: 使用1200p模型
    - 1250-1350px: 使用1300p模型
    - 1350-1450px: 使用1400p模型
    - 1450-1550px: 使用1500p模型
    - 1550-1750px: 使用1600p模型
    - 1750-1980px: 使用1920p模型
    - 大于1980px: 使用2048p模型

- **DAT2模型支持**：
  - 使用spandrel库加载DAT2模型
  - 支持更高质量的超分辨率效果

## 优化

### 1. 翻译器请求/响应改为JSON格式

- **OpenAI高质量翻译器**：
  - 修改系统提示词，要求模型返回JSON格式响应
  - JSON格式示例：`[{"id": 1, "translation": "翻译文本1"}, {"id": 2, "translation": "翻译文本2"}]`
  - 保留向后兼容性，支持解析非JSON格式响应

- **Gemini高质量翻译器**：
  - 同样修改为支持JSON格式响应
  - 统一翻译器接口，提高代码一致性

### 2. 翻译器Raw Response日志改为debug级别

- 将原始响应日志级别从info改为debug
- 减少正常使用时的日志输出量
- 保留调试能力，需要时可查看完整响应

### 3. MangaJaNai切割逻辑与RealCUGAN统一

- 统一使用相同的切片参数：
  - overlap=16
  - tile_size=400
- 提高代码一致性和可维护性
- 优化处理大图时的内存使用

### 4. launch.py支持--index-url参数

- 添加命令行参数`--index-url`
- 支持指定PyTorch安装的索引URL
- 优先级：命令行参数 > 环境变量 > 默认镜像
- 使PyTorch包使用官方源安装，提高安装成功率

### 5. 配置文件读取鲁棒性增强

- 优化配置文件解析逻辑
- 增强错误处理和异常捕获
- 提高程序稳定性

## 修复

- 增强模型下载的SHA256哈希验证功能

## 使用说明

### MangaJaNai超分模型

1. 在UI界面选择"MangaJaNai"作为超分模型
2. 选择倍率：x2、x4或DAT2 x4
3. 系统会根据图片分辨率自动选择最合适的模型

### 使用自定义PyTorch源

```bash
# 使用官方源安装
python packaging/launch.py --index-url https://download.pytorch.org/whl/cu121

# 使用镜像源安装
python packaging/launch.py --index-url https://mirrors.tuna.tsinghua.edu.cn/pytorch/whl/cu121/
```

## 技术细节

### MangaJaNai模型结构

MangaJaNai模型分为三类：
- **x2模型**：使用ESRGAN架构，2倍放大
- **x4模型**：使用ESRGAN架构，4倍放大
- **DAT2 x4模型**：使用DAT2架构，4倍放大，需要spandrel库支持

### JSON翻译格式

新的JSON翻译格式提供了更好的结构化数据，便于处理多行文本和保持翻译顺序。格式示例：

```json
[
  {"id": 1, "translation": "这是第一条翻译"},
  {"id": 2, "translation": "这是第二条翻译，\n包含换行符"}
]
```

### 模型哈希验证流程

1. 下载模型文件
2. 检查文件大小是否大于1KB
3. 计算SHA256哈希值
4. 与预设哈希值比对
5. 不匹配时提供详细错误信息
