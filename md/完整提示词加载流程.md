翻译请求 (source_lang, target_lang, custom_prompt, line_break_prompt, ...)
         │
         ▼
   _build_system_prompt()
         │
         ▼
┌─────────────────────────────────────┐
│ 1. 根据 source_lang 加载基础提示词   │
│    lang_to_prompt_file 映射表        │
│    JPN→ja.json, KOR→ko.json, ...    │
│    未匹配 → default.json            │
│    仍不存在 → system_prompt_hq.json │
└─────────────────────────────────────┘
         │
         ▼
┌─────────────────────────────────────┐
│ 2. 替换 {{{target_lang}}} 占位符    │
│    例: {{{target_lang}}} → "Simplified Chinese"
└─────────────────────────────────────┘
         │
         ▼
┌─────────────────────────────────────┐
│ 3. 组装最终提示词 (按顺序拼接)       │
│    ├─ retry_hint (如果是重试)       │
│    ├─ line_break_prompt (断句提示)  │
│    ├─ custom_prompt (用户自定义)    │
│    ├─ base_prompt (语言提示词)      │
│    └─ glossary_prompt (术语提取)    │
└─────────────────────────────────────┘
         │
         ▼
   返回 final_prompt → 作为 system_instruction 发送给 LLM
         │
         ▼
┌─────────────────────────────────────┐
│ 4. 构建 user_prompt (用户消息)       │
│    ├─ 上下文 (prev_context)         │
│    ├─ 待翻译文本 JSON               │
│    └─ 图片 (HQ模式)                 │
└─────────────────────────────────────┘
         │
         ▼
   调用 LLM API (OpenAI/Gemini)
         │
         ▼
   解析 JSON 响应 → 提取翻译结果
