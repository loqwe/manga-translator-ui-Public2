# 漫画网站广告过滤功能说明

## 1. 功能概述

漫画网站广告过滤功能是基于水印过滤系统的扩展，专门用于识别和过滤漫画网站中常见的广告内容。该功能可以有效去除漫画页面中的广告文本、网站宣传、版权声明等内容，提高翻译质量和阅读体验。

## 2. 实现原理

广告过滤功能通过以下方式实现：

1. **文本模式匹配**：使用部分匹配和完全匹配两种模式识别广告内容
2. **水印过滤系统集成**：复用现有的水印过滤机制，无需额外的处理流程
3. **规则文件配置**：通过 `watermark_filter.json` 文件进行规则配置，支持用户自定义

## 3. 过滤规则说明

广告过滤规则分为两类：

### 3.1 部分匹配规则 (partial_match_patterns)

部分匹配规则用于识别包含特定关键词的文本，不区分大小写。例如：

- `divascans` - 匹配包含 "divascans" 的任何文本
- `comick.io` - 匹配包含 "comick.io" 的任何文本
- `we are no longer uploading` - 匹配包含此短语的广告文本

### 3.2 完全匹配规则 (exact_match_patterns)

完全匹配规则要求文本完全相同才会被过滤，区分大小写。例如：

- `DIVASCANS.COM` - 仅匹配大写的 "DIVASCANS.COM"
- `COMICK.IO` - 仅匹配大写的 "COMICK.IO"
- `MANGA PARK` - 仅匹配大写的 "MANGA PARK"

## 5. 配置文件位置

广告过滤规则存储在以下位置：

```
examples/config/watermark_filter.json
```

## 6. 自定义过滤规则

用户可以通过编辑 `watermark_filter.json` 文件来添加自定义的广告过滤规则：

1. 打开 `examples/config/watermark_filter.json` 文件
2. 在 `partial_match_patterns` 数组中添加需要部分匹配的关键词
3. 在 `exact_match_patterns` 数组中添加需要完全匹配的文本
4. 保存文件后，程序会自动加载最新的过滤规则

示例：
```json
{
  "partial_match_patterns": [
    "existing_pattern",
    "your_new_pattern"
  ],
  "exact_match_patterns": [
    "existing_exact_match",
    "YOUR_NEW_EXACT_MATCH"
  ]
}
```

## 7. 过滤效果

广告过滤功能可以有效去除以下类型的内容：

1. 网站宣传横幅
2. 网站域名和链接
3. 版权声明和水印
4. 广告语句和宣传文本
5. 网站转移通知

## 8. 注意事项

1. 过滤规则过于宽泛可能会误过滤正常内容，请谨慎添加
2. 规则文件修改后会在5分钟内自动生效（缓存刷新时间）
3. 如需立即生效，请重启应用程序
